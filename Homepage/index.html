X-Host Chooser

X-Host Chooser is a "launch-pad" for Cygwin's XFree86 server XWin. It allows you to easily set all the options needed to connect to a remote workstation which allows remote logins via the XDMCP protocol.

X-Host Chooser is far from perfect and there are for sure quite some bugs in it - if you find any, please feel free to contact me at ASkwar@email-server.info! Feature suggestions are also welcome.

News:

  • 2003-03-15: Application internationalized. Currently it supports English and German. More languages are rather easy to add. Please contact me, if you'd like to translate it to another language.
  • 2003-03-13: Project started
